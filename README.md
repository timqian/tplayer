# Learn stream by build a music player

## Usage:

1. `npm install -g tplayer`
2. `tplayer dirToYour.mp3`

## Environment

`node v4.4.4` (not working on `v6.x.x`, because of [a bug of node-lame](https://github.com/TooTallNate/node-lame/issues/62))
